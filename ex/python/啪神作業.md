##### 程式設計題:完成等差數列的總和計算:1+5+9+13+ ...+81
```
total=n=0
while(n<81):
    n+=4
    total+=n
print(total)
```
##### 程式設計題:完成等差數列的積和計算:1乘5乘9乘13乘 ...乘33
```
total=n=0
while(n<33):
    n*=4
    total*=n
print(total)
```
